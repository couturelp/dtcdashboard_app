# PRD Part 07: Broader Web Research — Reviews, Ratings & User Sentiment

## Overview
Search the broader web for user reviews, ratings, and sentiment about BeProfit across all major review platforms. Capture what real users love and hate, common feature requests, churn reasons, and support quality feedback. This provides the user perspective that marketing and help docs won't reveal.

## Output Files
`beprofit-casestudy-info-111.md` through `beprofit-casestudy-info-130.md`

## Tasks

### Task 1: Shopify App Store Listing
- Search for BeProfit on the Shopify App Store
- Document: app name, ratings, review count, feature list, pricing as shown
- Capture screenshot descriptions from the listing
- Read recent reviews (positive and negative) — at least 20-30 reviews
- Categorize review themes (praise, complaints, feature requests)
- **Output:** `beprofit-casestudy-info-111.md`

### Task 2: Shopify App Store — Negative Reviews Deep Dive
- Focus specifically on 1-3 star reviews
- Categorize complaints by theme (bugs, missing features, pricing, support, accuracy)
- Note recurring pain points
- Capture developer/support responses to negative reviews
- **Output:** `beprofit-casestudy-info-112.md`

### Task 3: G2 Reviews
- Search "BeProfit review G2"
- Document overall rating, number of reviews, category rankings
- Capture pros and cons from multiple reviews
- Note company size and industry of reviewers
- **Output:** `beprofit-casestudy-info-113.md`

### Task 4: Capterra Reviews
- Search "BeProfit Capterra"
- Document ratings across categories (ease of use, features, support, value)
- Capture detailed review excerpts
- Note comparison mentions (what else reviewers considered)
- **Output:** `beprofit-casestudy-info-114.md`

### Task 5: Trustpilot Reviews
- Search "BeProfit Trustpilot"
- Document overall rating and review volume
- Capture review themes and notable excerpts
- **Output:** `beprofit-casestudy-info-115.md`

### Task 6: Other Review Sources
- Search for BeProfit reviews on Product Hunt, GetApp, Software Advice, etc.
- Check Reddit for BeProfit discussions (r/shopify, r/ecommerce, r/dropshipping)
- Capture any YouTube review videos and their key takeaways
- **Output:** `beprofit-casestudy-info-116.md`

### Task 7: User Sentiment Synthesis — Most Praised Features
- Aggregate across all review sources
- Rank the most commonly praised features
- Document specific quotes and evidence
- Note which user segments praise which features
- **Output:** `beprofit-casestudy-info-117.md`

### Task 8: User Sentiment Synthesis — Most Criticized Features
- Aggregate across all review sources
- Rank the most common complaints and pain points
- Document specific quotes and evidence
- Note severity and frequency of each issue
- **Output:** `beprofit-casestudy-info-118.md`

### Task 9: Common Feature Requests
- Aggregate feature requests from all review sources
- Rank by frequency and urgency
- Note which requests align with gaps found in help center research
- Document any feature requests that suggest market direction
- **Output:** `beprofit-casestudy-info-119.md`

### Task 10: Churn Reasons & Support Quality
- Document reasons users give for leaving or considering leaving
- Capture support quality feedback (response time, helpfulness)
- Note any migration patterns (what users switch to/from)
- **Output:** `beprofit-casestudy-info-120.md`

### Task 11: Pricing Sentiment
- Document user sentiment specifically about pricing
- Capture value-for-money perceptions
- Note which plans users are on and satisfaction levels
- Document any pricing complaints or praise
- **Output:** `beprofit-casestudy-info-121.md`

### Task 12: Checkpoint Review #7
- Perform structured checkpoint review
- Assess review coverage across platforms
- Identify any review sources not yet checked
- Synthesize overall user sentiment picture
- **Output:** `beprofit-casestudy-info-122.md`
