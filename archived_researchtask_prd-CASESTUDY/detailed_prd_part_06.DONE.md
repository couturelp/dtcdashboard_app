# PRD Part 06: Marketing Website — Features Pages, Pricing & Integration Pages

## Overview
Deep dive into BeProfit's feature-specific marketing pages, pricing page, and integrations marketing pages. These pages reveal how they package and position individual capabilities, their monetization strategy, and their integration ecosystem marketing.

## Output Files
`beprofit-casestudy-info-081.md` through `beprofit-casestudy-info-095.md`

## Tasks

- Identify all feature-specific pages on the marketing site
- Document how features are grouped and presented
- Note the feature page template/structure pattern
- **Output:** `beprofit-casestudy-info-081.md`

### Task 3: Features — Order & Product Analytics

- Capture positioning and benefit statements

### Task 4: Features — Ad Tracking & Attribution

### Task 5: Features — Custom Reports

### Task 6: Features — Remaining Feature Pages

### Task 7: Pricing Page — Full Documentation
- Document the pricing page in exhaustive detail
- Capture every plan name and price (monthly and annual)
- Document the complete feature matrix across all plans
- Note usage limits per plan (orders/month, stores, users, etc.)
- Capture any add-ons, enterprise tier, or custom pricing options
- **Output:** `beprofit-casestudy-info-087.md`

### Task 8: Pricing Psychology Analysis
- Use `marketing-psychology` skill on the pricing page
- Analyze anchoring, decoy effects, and tier positioning
- Note which plan is visually emphasized (recommended plan)
- Capture urgency/scarcity tactics on pricing page
- Document free trial CTA and terms
- Analyze annual vs monthly pricing differential strategy
- **Output:** `beprofit-casestudy-info-088.md`

### Task 9: Integrations Marketing Page
- Document the integrations overview marketing page
- Capture how integrations are categorized and presented
- Note which integrations are highlighted vs listed
- Compare marketing integration list vs help center integration details
- **Output:** `beprofit-casestudy-info-089.md`

### Task 10: Individual Integration Landing Pages
- Check if individual integrations have dedicated marketing pages
- Document any integration-specific landing pages (e.g., "BeProfit for Shopify")
- Capture integration-specific value propositions
- **Output:** `beprofit-casestudy-info-090.md`

### Task 11: Case Studies & Testimonials Pages
- Document dedicated case study or testimonials pages
- Capture customer names, industries, metrics shared
- Note the storytelling format used
- Analyze social proof effectiveness
- **Output:** `beprofit-casestudy-info-091.md`

### Task 12: Partner / Agency Pages
- Document any partner or agency program pages
- Capture partner benefits and requirements
- Note white-label or reseller options
- **Output:** `beprofit-casestudy-info-092.md`

### Task 13: Comparison Pages
- Find and document any "BeProfit vs X" comparison pages
- Capture competitive positioning claims
- Note which competitors they compare against
- Analyze comparison tactics and fairness
- **Output:** `beprofit-casestudy-info-093.md`

### Task 14: Demo, Tour & Other Landing Pages
- Document any product demo or tour pages
- Capture interactive demos, video walkthroughs
- Note any additional landing pages found in footer or sitemap
