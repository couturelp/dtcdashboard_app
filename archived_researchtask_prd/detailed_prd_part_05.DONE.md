# PRD Part 05: Marketing Website — Homepage, Positioning & Messaging

## Overview
Crawl and analyze the BeProfit marketing website (https://beprofit.co/) starting with the homepage and overall positioning. Use the `copywriting`, `marketing-psychology`, and `audit-website` skills to analyze messaging effectiveness, persuasion tactics, and site structure. Compare what they promise in marketing vs what the product actually does (based on help center research).

## Output Files
`beprofit-casestudy-info-071.md` through `beprofit-casestudy-info-080.md`

## Tasks

### Task 1: Homepage — Hero & Above the Fold
- Browse https://beprofit.co/
- Document the hero section (headline, subheadline, CTA, imagery)
- Capture above-the-fold content and first impression
- Analyze messaging using `copywriting` skill
- Note social proof elements visible immediately
- **Output:** `beprofit-casestudy-info-071.md`

### Task 2: Homepage — Full Page Analysis
- Document the entire homepage top to bottom
- Capture every section: features, benefits, testimonials, integrations, CTAs
- Note the narrative flow (how the page tells a story)
- Document all trust signals and social proof
- Capture footer content and navigation links
- **Output:** `beprofit-casestudy-info-072.md`

### Task 3: Marketing Psychology Analysis
- Use `marketing-psychology` skill to analyze the homepage
- Document persuasion tactics (social proof, authority, scarcity, urgency)
- Capture pricing psychology signals
- Note conversion optimization patterns
- Analyze CTA placement and messaging
- **Output:** `beprofit-casestudy-info-073.md`

### Task 4: Value Proposition Analysis
- Document BeProfit's primary value propositions (top 3-5)
- Capture how they differentiate from competitors
- Note target audience segments implied by messaging
- Analyze the "jobs to be done" they're addressing
- Compare marketing claims vs actual product capabilities (from help center)
- **Output:** `beprofit-casestudy-info-074.md`

### Task 5: Site Navigation & Structure
- Document the full site navigation (top nav, mega menus, footer)
- Use `audit-website` skill to evaluate site structure
- Map every page accessible from navigation
- Note any hidden pages or secondary navigation
- Capture sitemap.xml content (https://beprofit.co/sitemap.xml)
- **Output:** `beprofit-casestudy-info-075.md`

### Task 6: About / Company Page
- Document the About page content
- Capture company story, team info, mission statement
- Note any investor/funding mentions
- Document company size and growth signals
- **Output:** `beprofit-casestudy-info-076.md`

### Task 7: Blog & Content Marketing
- Scan the blog for recent posts (last 6-12 months)
- Document content themes and topics
- Note product announcements and feature launches
- Capture case studies or customer stories
- Analyze content marketing strategy and frequency
- **Output:** `beprofit-casestudy-info-077.md`

### Task 8: Legal Pages & Data Handling
- Skim Terms of Service for notable clauses
- Skim Privacy Policy for data handling practices
- Note any compliance certifications mentioned (SOC2, GDPR, etc.)
- Document data processing and third-party sharing details
- **Output:** `beprofit-casestudy-info-078.md`

### Task 9: Conversion Funnel Analysis
- Map the conversion funnel: homepage → features → pricing → signup
- Document each step in the funnel
- Note friction points and trust-building elements at each step
- Capture any exit-intent or engagement popups
- Analyze the overall conversion strategy
- **Output:** `beprofit-casestudy-info-079.md`

### Task 10: Checkpoint Review #5
- Perform structured checkpoint review
- Assess marketing website coverage
- Compare marketing claims vs product reality (from help center)
- Identify remaining marketing pages to visit
- **Output:** `beprofit-casestudy-info-080.md`

## Skills to Use
- `copywriting` — Analyze all marketing copy quality and tactics
- `marketing-psychology` — Identify persuasion patterns and behavioral triggers
- `audit-website` — Evaluate overall site structure and UX
- `agent-browser` — Browse all pages systematically
