# PRD Part 04: Help Center — Settings, Configuration, Billing & Troubleshooting

## Overview
Document all settings, configuration options, billing/plan details, and troubleshooting content from the Help Center. This covers account management, plan details and feature gating, and all FAQs and troubleshooting articles that reveal edge cases and product limitations.

## Output Files
`beprofit-casestudy-info-051.md` through `beprofit-casestudy-info-070.md`

- Document all account-level settings
- Capture profile management options
- Note email/notification preferences
- Document timezone, locale, and display settings
- **Output:** `beprofit-casestudy-info-051.md`

### Task 2: Store Configuration Settings
- Document per-store configuration options
- Capture currency settings and conversion preferences
- Note default cost and margin settings
- Document any store-level customization
- **Output:** `beprofit-casestudy-info-052.md`

### Task 3: Billing, Plans & Subscriptions
- Document all billing-related help articles
- Capture plan names, pricing, and feature differences
- Note upgrade/downgrade flow
- Document payment methods accepted
- Capture cancellation process and data retention policy
- **Output:** `beprofit-casestudy-info-053.md`

### Task 4: Plan Feature Gating Details

- Capture usage limits per plan (orders/month, stores, users, integrations)

- Capture any free tier details if one exists
- **Output:** `beprofit-casestudy-info-054.md`

### Task 5: Data Management & Privacy
- Document data handling practices mentioned in help docs
- Capture data deletion and export capabilities
- Note GDPR or privacy compliance mentions
- Document data retention policies
- **Output:** `beprofit-casestudy-info-055.md`

### Task 6: Troubleshooting — Data Sync Issues
- Document all data sync troubleshooting articles
- Capture common sync errors and resolutions
- Note data discrepancy explanations (why numbers don't match Shopify, etc.)
- Document manual data correction options
- **Output:** `beprofit-casestudy-info-056.md`

### Task 7: Troubleshooting — Integration Connection Issues
- Document integration-specific troubleshooting
- Capture reconnection procedures
- Note permission and access issues
- Document platform-specific known issues
- **Output:** `beprofit-casestudy-info-057.md`

### Task 8: Troubleshooting — Reporting & Calculation Issues
- Document reporting accuracy troubleshooting
- Capture explanations for common calculation discrepancies
- Note edge cases in profit calculation (refunds, chargebacks, gift cards)
- Document workarounds for known limitations
- **Output:** `beprofit-casestudy-info-058.md`

### Task 9: FAQ Sections
- Document all FAQ content
- Capture frequently asked questions and their answers
- Note what questions reveal about common user confusion
- Identify product gaps implied by FAQ content
- **Output:** `beprofit-casestudy-info-059.md`

### Task 10: Help Center — Remaining Articles
- Sweep through any help center articles not yet covered
- Document any remaining categories or standalone articles
- Capture articles about new features, changelog/updates
- Note any beta or upcoming features mentioned
- **Output:** `beprofit-casestudy-info-060.md`

### Task 11: Help Center Information Architecture Analysis
- Analyze the overall structure and organization of the help center
- Document how topics are categorized and cross-linked
- Note search functionality and discoverability patterns
- Assess completeness and quality of documentation
- **Output:** `beprofit-casestudy-info-061.md`
